# scratch/design-lab/

Visual design explorations — HTML mockups, Figma exports, color/type studies, anything where you want to see options side-by-side before committing.

## Conventions

- **Group by project subdir**, not by date: `design-lab/<project-or-feature>/`
- One HTML file per variation when comparing options (e.g., `option-a.html`, `option-b.html`)
- Drop a one-line `NOTES.md` in each subdir explaining what you were exploring
- Promote winners out of `design-lab/` into the actual project; archive the rest under `scratch/archive/`

## Examples

```
design-lab/
  homepage-redesign/
    option-a-bento.html
    option-b-stacked.html
    NOTES.md
  brand-color-trial/
    palette.html
```
