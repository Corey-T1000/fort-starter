# scratch/archive/

Finished or paused work moved out of active scratch. Keeps the working dirs (`design-lab/`, `playground/`, `research/`, `scripts/`) focused on what's currently in motion.

## Conventions

- Move completed or abandoned subdirs here as-is — preserve the original folder name
- Optional: prefix with a date when you archived it, e.g., `archive/2026-04-22-old-homepage-mocks/`
- Don't reorganize on archive — just relocate. The point is low friction.
- Periodically prune (every few months) — anything older than ~6 months that you haven't reached for is probably safe to delete

## Examples

```
archive/
  2026-03-shipped-onboarding-flow/
  abandoned-graphql-experiment/
  old-brand-explorations/
```
